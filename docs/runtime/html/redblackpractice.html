1 Practice Artefact for Red Black Tree

1.1 Features

  • User can insert,search ,delete a node in a red black tree
  • User can visualize the movement of nodes clearly along with viewing the required steps
  • User can control the speed of the animation

  • The required animation libraries are imported
  • The required artefact javascript files are imported.

2.4 Instruction Box

  • The instruction box above the artefact carries information about how to use the artefact.
  • It has details over how the controls presented on the screen work.
  • A brief description of the usage of artefacts is also provided here. 
        <body onload="init();" class="VisualizationMainPage">

        <div class="instruction-box">
                <button class="collapsible">Instructions</button>
                <div class="content">
                  <b>Instructions</b>: 
                  <ul>
                           <li>Click generate root to generate the root of the tree</li> 
                          <li> Check the value to be inserted next at the bottom</li>
                          <li> Choose the next step by clicking the appropriate button</li>
                          <li> Check  validity of the step in the comments section at bottom</li>
                          <li> Click show steps to check all the steps</li>
                          <li>You can decrease the <b>speed</b> of the animation by using the <b>animation slider</b>.</li>
                          <li>It is recommended that you set the speed to a minimum value to visualize each step properly.</li>

                          <li>You can use the <b>Pause</b> button to pause the animation at any point of time.</li>
                          <li>After animation has been paused, click on the <b>Step Forward</b> and <b>Step Backward</b> button to step through the animation.</li>
                          <li>The <b>Skip Forward</b> and <b>Skip Backward</b> buttons can be used to undo and redo operations after they have been performed on the tree.</li>
                  </ul> 
                </div>
        </div>

                var collapsibleEl = document.getElementsByClassName("collapsible")[0];
                collapsibleEl.addEventListener("click", function() {
                        this.classList.toggle("active");
                        var content = this.nextElementSibling;
                        if (content.style.maxHeight){
                                content.style.maxHeight = null;
                        } else {
                                content.style.maxHeight = content.scrollHeight + "px";
                        }
